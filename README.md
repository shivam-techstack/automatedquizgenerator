# 📚 Automated Quiz Generator

An AI-powered web application that automatically generates quizzes from user-provided input. This project is designed to help students, teachers, and learners quickly create multiple-choice quizzes and test knowledge in an interactive way.

---

## 🚀 Features

* 🧠 Generate quizzes from text or topic input
* ❓ Automatic Multiple-Choice Question (MCQ) creation
* 📊 Instant score calculation
* 🎯 Interactive and user-friendly interface
* ⚡ Fast and lightweight web application
* 🌐 Frontend using HTML, CSS, JavaScript
* 🐍 Backend powered by Python (Flask)

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/skroy3300-art/automatedquizgenerator.git
cd automatedquizgenerator
```

---

### 2️⃣ Create Virtual Environment (Recommended)

```bash
python -m venv venv
```

Activate the environment:

**Windows:**

```bash
venv\Scripts\activate
```

**Mac/Linux:**

```bash
source venv/bin/activate
```

---

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 4️⃣ Run the Application

```bash
python app.py
```

---

### 5️⃣ Open in Browser

```
http://127.0.0.1:5000/
```

---

## 🧠 How It Works

1. User enters a topic or input text
2. Backend processes the content
3. Quiz questions are generated dynamically
4. Questions are displayed on the frontend
5. User selects answers
6. System calculates and displays the score instantly
